# Assignments

## Daum 이미지 검색 api를 활용한 이미지 검색앱 만들기

Debounce를 활용해 모든 입력이 끝난 후 1초 뒤 검색 요청

새로운 키워드가 입력될 때 마다 api요청

검색 결과가 없을 시, 배너를 활용한 안내

검색창이 빈 값인 경우, 배너를 띄우지 않음


## 구현 방법
검색의 경우 일부 RxSwift 사용

Scroll시 업데이트 하는 경우 Delegate 패턴 사용
